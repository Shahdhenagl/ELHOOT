# 🐋 نظام إدارة المحل - الحوت

نظام شامل لإدارة الإيرادات والمصروفات والمحافظ المالية

## المميزات:

✅ إضافة الإيرادات والمصروفات
✅ إدارة الوسائل المالية (نقدي، محفظة، بطاقات)
✅ تحويلات بين الوسائل
✅ فلترة المعاملات حسب الشهر والوسيلة
✅ تقارير شهرية وسنوية
✅ حساب الربح والخسارة
✅ تصدير PDF و CSV

## الاستخدام:

1. افتح ملف ccounting_system.html بالمتصفح
2. ابدأ بإضافة الوسائل المالية
3. سجل المعاملات يومياً
4. اطلع على التقارير الشهرية

جميع البيانات تُحفظ محلياً على جهازك!
